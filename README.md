# Regression-model
How to Run the Code

1. Ensure Python 3.x is installed on your system.

2. Install the required libraries in your Python environment:
    pandas
    numpy
    scikit-learn

3. Place the dataset file house_data.csv in the same directory as the script file reg.py.

4. Open the project folder in your Python IDE (such as VS Code or PyCharm).

5. Open the file reg.py.

6. Run the script using your IDE’s run option.

7. After execution, the program will:

    Load the dataset

    Perform preprocessing

    Train the regression model

    Display MSE and R² score
